Who we are

We are a group of six students from eMobillis, currently training in Django web development. Our project is a collaborative effort to build a dynamic web application using Django for the back-end and Bootstrap for the front-end.

  • Building a responsive and modern web interface with Bootstrap.
  • Leveraging Django's powerful back-end capabilities for a seamless experience.
  • Enhancing team collaboration and coding best practices.

Our project integrates the flexibility of Django for dynamic content and the responsive design features of Bootstrap to ensure a user-friendly, mobile-first application. Through this assignment, we aim to strengthen our skills in both front-end and back-end development.
